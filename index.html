Welcome to HTML and CSS

HTML（HyperText Markup Language）はウェブページを構築するための基本的な言語です。

CSS（Cascading Style Sheet）はHTMLと組み合わせて使用することで、Webサイトの色やサイズ、
レイアウトなどのデザインを定義するための言語です。
